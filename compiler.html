Phase 4 — Intermediate Code

Three-Address Code Generation.

; ── Function: main ── FUNC_BEGIN main t1 = 4 day = t1 t2 = day IF t2 == 1 GOTO L1 IF t2 == 2 GOTO L2 IF t2 == 3 GOTO L3 IF t2 == 4 GOTO L4 IF t2 == 5 GOTO L5 IF t2 == 6 GOTO L6 IF t2 == 7 GOTO L7 GOTO LDEFAULT L1: PARAM "Sunday\n"; CALL printf, 1; GOTO LEND L2: PARAM "Monday\n"; CALL printf, 1; GOTO LEND L3: PARAM "Tuesday\n"; CALL printf, 1; GOTO LEND L4: PARAM "Wednesday\n"; CALL printf, 1; GOTO LEND L5: PARAM "Thursday\n"; CALL printf, 1; GOTO LEND L6: PARAM "Friday\n"; CALL printf, 1; GOTO LEND L7: PARAM "Saturday\n"; CALL printf, 1; GOTO LEND LDEFAULT: PARAM "Invalid\n"; CALL printf, 1; GOTO LEND LEND: retval = 0; RETURN FUNC_END main

Phase 5 — Code Optimization

Improving efficiency.

P1: Constant Propagation
t1 = 4; day = t1; t2 = day;
day = 4; t2 = 4;
P2: Dead Code Elimination

GOTO L4; (Pruned 7 IFs)
P3: Temp Elimination
t2 = 4; IF t2==4 GOTO L4;
GOTO L4;
P4: Jump Threading
GOTO L4; ... GOTO LEND;
Inlined RETURN.

Final Optimized TAC
